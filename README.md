## 🤖 AI Projects

- **[Proof of Engagement](https://github.com/pwilson77/proof-of-engagement)** — Trustless on-chain settlement for autonomous agent task execution on Solana. Creator Agents post campaigns with USDC escrow; Validator Agents score evidence via pluggable adapters; ConsensusOrchestrator releases payment when quorum is reached. Supports Direct and RFQ (bid-based) campaign modes, with MagicBlock Ephemeral Rollups for ~50ms validator scoring.

- **[Agentic Dark Matter](https://github.com/pwilson77/agentic-dark-matter-oracle)** — A2A (agent-to-agent) settlement infrastructure. Enables autonomous agents to create escrow agreements, approve settlements, and release funds trustlessly — without human intermediaries. Built on Solana with a TypeScript SDK (`@adm/agent-sdk`) for the full escrow lifecycle.

- **[TAMP](https://github.com/pwilson77/tamp)** — Task & Agent Management Protocol. Structured task routing and coordination layer for multi-agent pipelines, letting agents publish, claim, and verify work across heterogeneous systems.
